Installation - Expand zip file to work location.

Setup Instructions

Tomcat: Install latest version of Tomcat and get it running. Add the following to $TOMCAT_HOME/conf/context.xml

   <Resource name="jdbc/Autos" auth="Container"
             type="javax.sql.DataSource"
             username="autouser"
             password="autopass"
             driverClassName="com.mysql.jdbc.Driver"
             url="jdbc:mysql://localhost/autos"
             maxActive="10" maxIdle="4"/>

Ant: Install latest version of Ant and get it running

MySQL: Install latest version of MySQL and get it running. Run the auto-sample.sql script to create the database, users, tables, and initial data. (Login to mysql as root user then run: mysql>source {path/to/autos/auto-sample.sql)

Running Instructions - WEB

Start by Listing all of the autos

Editing Instructions - Use eclipse or any editor.

Controllers are defined in web.xml:
  • list - ca.bob.autos.web.AutoLister
  • lookup - ca.bob.autos.web.AutoLookup
  • update - ca.bob.autos.web.AutoUpdate
  • delete - ca.bob.autos.web.AutoDelete

Build instructions. Use Eclipse to build, or run ant from command like from the autos folder:

Modify build.properties to suit your needs. This one is for Windows. May need to make changes for linux and mac. Build process will compile all files, create a war file and copy the war file to the Tomcat directory as defined in build.properties. War file is in the 'dist' folder for deployment elsewhere.

Run from command line.

Open a command box and move to the autos work folder. Type 'run' on Windows or './run.sh' on unix. Examples:

autos>run list
autos>run update -1 Honda Civic Grey 1000000   (creates new record) 
autos>run update 4 Honda Civic Red 1200000 (updates record with id=4)
